# Product Tracker

A Symfony 7 web application for managing data with full CRUD functionality.

## Features:

- Create, read, update, and delete items
- Data persistence using Doctrine ORM
- SQLite database integration
- Form handling and validation
- Structured Symfony architecture (Controllers, Services, Repositories)

## Tech Stack:

- PHP 8.x
- Symfony 7
- Doctrine ORM
- SQLite
- Twig

## Architecture:

The application follows Symfony best practices:
- Separation of concerns (Controller / Service / Repository)
- Dependency Injection
